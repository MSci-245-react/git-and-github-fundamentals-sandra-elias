## The Two Special Values of JavaScript types
In lecture 1, we discussed the two special "nothing" values in JavaScript.
I learned that `null` means you explicitly set no value, and `undefined` means a value hasn't been assigned yet.

```javascript
typeof undefined
typeof {}
```

```
'undefined'
'object'
```

- `null` is set by you
- `undefined` is the default value when it hasn't been assigned yet
